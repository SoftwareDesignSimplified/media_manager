# Media Manager

## What

A small app written in Ruby to manage your media.

Allows you to add books, games and music.

Stores data locally in JSON documents.

## Setup

```
brew install asdf
asdf install ruby 3.2.2
```

## Run

```
bin/run
```

This will run the app.

## Test

```
bin/test
```

This runs the unit tests.
